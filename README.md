
## 📖 Overview

The Employee Management System is a full-stack web application developed to simplify employee data management. It allows users to perform complete **CRUD (Create, Read, Update, Delete)** operations with a clean and user-friendly interface.

The frontend is built using **React.js** and communicates with a **Spring Boot** backend through REST APIs. Employee data is securely stored in a **MySQL** database using Spring Data JPA. This project demonstrates modern full-stack development practices and follows a clean architecture for easy maintenance and scalability.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Employee-Management-System.git
```

### Backend Setup

```bash
cd ems-backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd ems-frontend
npm install
npm run dev
```

## 📡 REST API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /api/employees | Get All Employees |
| GET | /api/employees/{id} | Get Employee By ID |
| POST | /api/employees | Add Employee |
| PUT | /api/employees/{id} | Update Employee |
| DELETE | /api/employees/{id} | Delete Employee |
